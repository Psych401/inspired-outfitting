# Background Images

This directory contains background images for the virtual try-on feature.

## Required Images

- `studio.jpg` - Professional photo studio background
- `fitting-room.jpg` - Virtual dressing room environment
- `studio-thumb.jpg` - Thumbnail for studio background (200x200px recommended)
- `fitting-room-thumb.jpg` - Thumbnail for fitting room background (200x200px recommended)

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 1920x1080px or larger
- **Aspect Ratio**: 16:9 or flexible
- **File Size**: Optimize for web (under 500KB each)
- **Thumbnails**: 200x200px for previews

## Notes

- White background is generated programmatically (no image needed)
- If background images are missing, the system falls back to white background
- Images should be high quality but optimized for fast loading
